## Gotcha: onnxruntime-gpu CUDA 13 vs môi trường CUDA 12 (libcudart.so.13)

**Triệu chứng**
```
Cannot import custom_nodes/WanAnimatePreprocess module for custom nodes:
libcudart.so.13: cannot open shared object file: No such file or directory
```
(hoặc node bất kỳ import `onnxruntime` đều fail tương tự).

**Nguyên nhân gốc**
- Từ `onnxruntime-gpu==1.23.0` trở lên, wheel build cho **CUDA 13**
  (`requires_dist` có `nvidia-cuda-runtime-cu13~=13.0`) → cần `libcudart.so.13`.
- Môi trường chạy dùng **CUDA 12**: `torch==2.10.0` (wheel cu12x mặc định),
  `cupy-cuda12x`. Không có `libcudart.so.13` → import onnxruntime chết.
- Nếu `pip install onnxruntime-gpu` **không pin version** → luôn kéo bản mới nhất
  (1.27+ CUDA 13) → lỗi.

**Cách fix (đã áp dụng)**
- Ghim `onnxruntime-gpu==1.22.0` (bản CUDA 12 **cuối cùng**) ở mọi chỗ trong repo:
  - `requirements.txt`
  - `custom_nodes.py` (list `extra_pkgs` + fix-section cuối)
  - `install_nodes.py` (fix-section cuối)
- Fix-section chạy **cuối cùng** trong cả 2 script cài → last-wins, ép đúng 1.22.0
  kể cả khi node khác (WanAnimatePreprocess, fantasyportrait, comfy_mtb) khai báo
  `onnxruntime-gpu` không pin.

**Remediation cho môi trường ĐANG lỗi (đã cài 1.27)**
```
pip uninstall -y onnxruntime onnxruntime-gpu
pip install onnxruntime-gpu==1.22.0
```
Rồi restart ComfyUI.

**Nếu tương lai nâng lên CUDA 13** (torch cu13x): lúc đó mới bỏ pin / cho phép
onnxruntime-gpu >= 1.23.0. Đừng nâng lẻ onnxruntime khi torch vẫn là CUDA 12.
